Вопрос 1. Целью создания АИУС является:

Управление производственными процессами без участия человека

Вопрос 2. Закон необходимого разнообразия в кибернетике говорит о том, что:

Для управления сложными объектами с большим разнообразием необходима управляющая система с вдвое большим разнообразием, чем объект

Для управления сложными объектами с большим разнообразием достаточна управляющая система с меньшим разнообразием, чем объект

Для управления сложными объектами с большим разнообразием необходима управляющая система с не меньшим разнообразием, чем объект
